Tentang MiFarKomp

MiFarKomp (Mikholas & Farikhin Kompresi) adalah sebuah proyek aplikasi web yang dirancang untuk memudahkan pengguna dalam mengkompresi berbagai jenis file secara online.

Aplikasi ini dibangun menggunakan Python (Flask) untuk backend dan Tailwind CSS untuk antarmuka yang bersih dan responsif.
